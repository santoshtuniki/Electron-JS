Electron applications are structure with 2 processes:

Main Process

  • Only 1 main process for an application
  • Responsible for managing application lifecycle
  • Runs in Node.js environment

Renderer Process

  • Each Renderer Process corresponds to a web page or view in your Electron application.
  • Renderer Processes are sandboxed, which means they have limited access to Node.js APIs to enhance security.
  • Each BrowserWindow created in the Main Process has its own Renderer Process.
